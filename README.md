# App to enter student scores


## Instructions to run the app

* Clone the git repo
```
git clone https://github.com/hjerger/student-scores-app.git
```  

* Install node_modules (tested with node v14.5.0, npm 6.14.5)
```
npm install  
```

* Start the app
```
npm start  
```

* Application running on port 3000
```
http://localhost:3000
```
